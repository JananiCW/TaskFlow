<h1 align = "center"> TaskFlow — Project Management Tool </h1>

A full-stack project management tool built for software engineering teams using **React**, **Go (Gin)**, and **MongoDB**.

---

## 🌟 Features

- 🚀 Create and manage multiple projects
- 📋 Kanban board — To Do, In Progress, Done
- 📌 Add tasks with priority and due dates
- 👥 Add and manage team members
- 📊 Progress tracker per project
- 🗑️ Delete projects and tasks

---

## 🧱 Tech Stack

| Part       | Technology        |
|------------|-------------------|
| Frontend   | React             |
| Backend    | Go + Gin          |
| Database   | MongoDB (Local)   |
